# VCop2 explorer

> ⚠️I won't provide the game files for you. You can still download them from [myabandonware](https://www.myabandonware.com/game/virtua-cop-2-byi#download) for example

https://jevarg.github.io/re-virtua-cop-2/

You can run the explorer locally with the following command:
```sh
pnpm dev
```
